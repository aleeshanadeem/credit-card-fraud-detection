# 💳 Credit Card Fraud Detection using Machine Learning

##  Project Overview

Credit card fraud is one of the most significant challenges faced by financial institutions. Detecting fraudulent transactions accurately can help reduce financial losses and improve customer security.

This project applies machine learning techniques to identify potentially fraudulent credit card transactions based on transaction-related features.

---

##  Objectives

- Analyze transaction patterns.
- Explore feature relationships through EDA.
- Engineer meaningful features.
- Train machine learning models for fraud detection.
- Evaluate model performance using classification metrics.

---

## 📂 Dataset Information

The dataset contains transaction-related features:

| Feature | Description |
|----------|-------------|
| Time | Time elapsed between transactions |
| V1-V5 | Anonymized transaction features |
| Amount | Transaction amount |
| Class | Target Variable (0 = Normal, 1 = Fraud) |
